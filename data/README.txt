Mijn data bestaat uit een json file en geojson file die samen 1 geosjon file vormen.

ikgeoData.json : Dit is de data die ik van de website van cbs heb gehaald voor al mijn benodigde data.

rotterdam_buurten.geojson : Dit is de geojson data waarbij ik de kaart van Rotterdam heb met alle buurten erin. Deze heb ik van https://erikmeerburg.carto.com/tables/rotterdam_buurten/public/map

rotterdamGeoJSON: Dit is de gecombineerde eindresultaat tussen ikgeoData en rotterdam_buurten. Met deze file heb ik alle data van cbs als geojson Rotterdam kaart beschikbaar.
